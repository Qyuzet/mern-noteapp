Key Endpoints

GET /api/products
Get all tasks
POST /api/products
Create a new task (requires authentication)
PUT /api/products/:id
Update a task (requires authentication)
DELETE /api/products/:id
Delete a task (requires authentication)
POST /api/users
Register a new user
POST /api/users/login
Login and get authentication token
